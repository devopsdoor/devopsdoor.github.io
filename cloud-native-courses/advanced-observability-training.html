RoadMap

Your Learning Curve.

  • Week 1 - Till Day 5

    Pre-Requisites

    A practical training that covers all the basics you need. It covers

    microservices (React, JAVA and Python),
    building and running them on cloud VMs,
    containerizing and deploying them using Docker,
    Running and Orchestrating them on Kubernetes etc !

  • Week 2 - Day 1

    "MASTERING OBSERVABILITY FUNDAMENTALS"

    • Discover the key concepts behind Observability, Monitoring, Logging and Tracing

    • Learn how to set up a generic monitoring infrastructure using Prometheus

    • Explore PromQL and the metrics format to extract valuable insights

  • Week 2 - Day 2

    "USING PROMETHEUS EXPORTERS TO MONITOR MYSQL"

    • Understand the importance of monitoring databases and the benefits of using Prometheus exporters

    • Learn how to set up a MySQL server on AWS and configure the MySQL exporter

    • Discover how to scrape metrics and visualize them using Prometheus Dashboard and Grafana

  • Week 2 - Day 3

    "VISUALIZING COMPLEX INFRASTRUCTURE WITH GRAFANA"

    • Dive into the world of complex infrastructures and find out what an SRE wants

    • Explore PromQL's query examples and arithmetic operations to generate useful reports

    • Learn how to set up Node exporter, calculate average CPU usage, and install Grafana on AWS

  • Week 2 - Day 4

    "CREATING EFFECTIVE GRAFANA DASHBOARDS"

    • Master the art of creating effective Grafana Dashboards that communicate meaningful insights

    • Discover how to add panels and organize dashboards into folders

    • Learn how to import existing dashboards, explore them and save them in JSON format

  • Week 2 - Day 5

    "LEVERAGING PROMETHEUS RULES AND ALERTS"

    • Learn how to create recording and alerting rules in Prometheus to monitor CPU and disk utilization

    • Discover how to set up Alert Manager and consolidate alerts to different channels

    • Explore the recording and alerts from Grafana to ensure you stay on top of issues before they become critical

  • Week 3 - Day 6

    "PUSHING METRICS AND ALERTING WITH PUSH GATEWAYS"

    • Discover a scenario where you need to push metrics and how to set up push gateway to do it

    • Explore Alert Manager's role in sending consolidated alerts to different channels

    • Learn how to configure Prometheus to scrape push gateway and publish metrics from a script

  • Week 3 - Day 7

    "EFFECTIVE CENTRALIZED LOGGING WITH GRAFANA LOKI"

    • Learn the importance of centralized logging and the tools used for logging

    • Explore the architecture of Grafana Loki, set it up and run its components as Docker containers

    • Discover how to visualize logs in Grafana and communicate between services using labels

  • Week 3 - Day 8

    "RUNNING PROMETHEUS ON KUBERNETES"

    • Learn why Prometheus is the de facto standard for Kubernetes and its benefits

    • Discover how to install Prometheus using Kubernetes Operator, Helm and Helm Chart

    • Explore default components, dashboards, exporters and targets in Grafana and Prometheus

  • Week 3 - Day 9

    "CUSTOMIZING PROMETHEUS CONFIGURATIONS FOR YOUR NEEDS"

    • Learn how to customize Prometheus configurations using Configmaps and Secrets

    • Discover how to manage default configuration changes, edit using Kubectl and self-manage

    • Explore ways of adding new targets to scrape, ServiceMonitors and monitoring external resources outside Kubernetes

  • Week 3 - Day 10

    "MEASURING SERVICE-LEVEL INDICATORS (SLI) AND OBJECTIVES (SLO)"

    • Explore what customers want and how to use Prometheus to measure latency and availability

    • Learn how to define SLI, SLO and use Sloth to generate recording and alerting rules in Prometheus

    • Discover the benefits of OpenSLO and its use cases

  • THE ULTIMATE OBSERVABILITY PROJECT-

    1 WEEK


    • Now you are done with the course and it's time to start the most amazing part of this course. 20+ engineers working together to implement the best Observability practices on a project, to make you ready for your Job search. The project is too big and wide that if one person handles it in a company, it takes 3 to 4 months. You will also go through an interview and resume preparation session at the end to kickstart your Job hunt!
