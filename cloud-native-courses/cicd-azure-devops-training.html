Why This Course?

This course covers the fundamentals of CI/CD with Azure DevOps and provides hands-on experience with building CI/CD pipelines for software development projects. Participants will learn about source code management, developer workflows, Azure Boards, and Azure Pipelines. By the end of the course, students will have completed an advanced CI/CD project and gained the confidence to apply their knowledge in interviews and real-world scenarios.

RoadMap

Your Learning Curve.

  • Week 1 - Till Day 5

    Pre-Requisites

    A practical training that covers all the basics you need. It covers

    microservices (React, JAVA and Python),
    building and running them on cloud VMs,
    containerizing and deploying them using Docker,
    Running and Orchestrating them on Kubernetes etc !

  • Week 2 - Day 1

    CI/CD Fundamentals and Source Code Management with GitHub

    Learn about Continuous Integration/Continuous Delivery (CI/CD) and why it's important for software development. Explore the different stages of CI/CD and how source code management fits in. Discover the popular tool GitHub for managing your source code and create a repository.

    • Introduction to CI/CD and its importance

    • Key stages in a CI/CD pipeline

    • Source code management and its importance

    • Choosing the right code management tool for your project

    • Getting started with GitHub for source code management

    • Setting up a GitHub repository for your project

  • Week 2 - Day 2

    Developer Workflow and Branching Strategies

    Explore the developer workflow and how to manage branches in your Git repository. Learn about best branching strategies and mainline development. Practice creating a pull request.


    • Understanding developer workflows and their importance

    • Best practices for branching strategies

    • Introduction to the Mainline branching strategy

    • Implementing developer workflows using branches

    • Creating a pull request in GitHub for code review

  • Week 2 - Day 3

    Continuous Integration, Code Review, and Merging

    Discover the importance of code reviews and learn how to merge code changes to the mainline. Dive into Continuous Integration (CI) and learn how to track your work using Azure DevOps. Create an organization and project in Azure DevOps.

    • Continuous Integration and why it's important

    • How to track changes and collaborate effectively

    • Code review and its importance in maintaining code quality

    • Merging changes to the mainline

    • Introduction to Azure DevOps and creating an organization and projects

  • Week 2 - Day 4

    Agile Project Management with Azure Boards

    Learn about Azure Boards, a powerful tool for work item management. Practice creating work items, planning sprints, and exploring developer workflows. Discover best practices for using Azure Boards.


    • Introduction to Azure Boards for project management

    • Creating work items and planning sprints

    • Overview of the developer workflow in Azure Boards

    • Best practices for using Azure Boards effectively

  • Week 2 - Day 5

    Automating Developer Workflow with Azure Pipelines

    Explore the importance of automating your developer workflow and learn about pipelines. Dive into Azure Pipelines, a popular tool for automating software builds and deployments.


    • Integrating developer workflow with Azure Boards

    • Understanding the need for automating developer workflow

    • Introduction to Azure Pipelines for continuous delivery

    • Testing the developer workflow using Azure Pipelines

  • Week 3 - Day 6

    Customizing Azure Boards and Pipelines

    Learn how to integrate Azure Boards with GitHub and customize boards to fit your team's needs. Discover how to visualize blocked items and customize columns and work progress. Practice working with pipelines.


    • Integrating Azure Boards with GitHub Projects

    • Customizing and styling Azure Boards

    • Visualizing blocked items and customizing columns

    • Overview of Azure Pipelines and creating pipelines

  • Week 3 - Day 7

    Running and Managing Pipelines with Azure Pipelines

    Learn about running and managing pipelines with Azure Pipelines. Discover when your pipeline will run and how to manage it. Create a CI pipeline for your frontend project.


    • Where to run your pipelines and how to manage them

    • Setting up Azure Pipelines for your project

    • Overview of the CI pipeline for frontend projects

  • Week 3 - Day 8

    CI/CD for Your Application with Azure Pipelines

    Explore how to use Azure Pipelines to set up CI/CD for your application. Learn about building Docker images from source code, preparing a Kubernetes cluster and container registry, creating a pipeline to build, push, and deploy, and checking the status of the pipeline build. Discover the VM that runs your pipeline and how to monitor pipeline status.


    • Preparing the Dockerfile to build images from source code

    • Setting up Kubernetes clusters and container registries

    • Creating pipelines to build, push, and deploy applications

    • Checking the status of pipeline builds and using VMs for pipelines

  • Week 3 - Day 9

    Advanced Pipelines with Azure Pipelines

    Learn about pipeline syntax and how to use variables in your pipelines. Discover how to manage dependencies and connect to external services. Explore using Docker Hub and GKE for your pipelines and learn about permissions on GKE Cluster - RBAC.


    • Pipeline syntax and using variables

    • Conditional execution of jobs and managing dependencies

    • Connecting to external services and using Docker Hub and GKE

    • Setting permissions on GKE clusters with RBAC

  • Week 3 - Day 10

    Continuous Delivery and Best Practices

    Explore local, dev, and staging environments and learn about continuous delivery. Discover best practices for CI/CD and release management.


    • Understanding the benefits of continuous delivery

    • Setting up local, dev, and staging environments

    • Best practices for CI/CD pipelines and continuous delivery

  • Days 11-15

    Advanced CI/CD Project

    Participate in an advanced CI/CD project with 20+ IT professionals. Practice using industry best practices and gain confidence to ace interviews.


    • An advanced CI/CD project involving 20+ IT professionals

    • Best industry practices for CI/CD projects

    • Gaining confidence to take on interviews with industry-standard CI/CD skills

    Your Success
