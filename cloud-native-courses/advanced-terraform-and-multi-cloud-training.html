RoadMap

Your Learning Curve.

  • Week 1 - Till Day 5

    Pre-Requisites

    A practical training that covers all the basics you need. It covers

    microservices (React, JAVA and Python),
    building and running them on cloud VMs,
    containerizing and deploying them using Docker,
    Running and Orchestrating them on Kubernetes etc !

  • Week 2 - Day 1

    Introduction to Cloud Computing and Infrastructure as Code

    Gain an understanding of cloud computing and Infrastructure as Code (IaC) principles. Explore the benefits and key concepts of working with cloud providers such as AWS, Google Cloud, and Azure. Set up your development environment and learn to configure Terraform for seamless infrastructure management.


    • Overview of Cloud Computing: Public cloud providers (AWS, Google Cloud, Azure), benefits, and key concepts.

    • Introduction to Infrastructure as Code (IaC): Benefits, principles, and comparison with traditional infrastructure management

    • Introduction to Terraform: Overview of Terraform, its features, and advantages

    • Setting up the development environment: Installing Terraform and configuring providers (AWS, Google Cloud, Azure).

  • Week 2 - Day 2

    Getting Started with Terraform and AWS

    Dive into AWS and Terraform integration. Learn the essentials of Terraform Configuration Language (HCL) and build your first infrastructure components, including EC2 instances, VPCs, and security groups. Develop a solid foundation in deploying and managing resources on AWS using Terraform.


    • Introduction to AWS: Overview of key AWS services and concepts (EC2, S3, VPC, etc.)

    • Understanding Terraform Configuration Language (HCL): Syntax, variables, and modules

    • Terraform Basics: Provider configuration, resources, and data sources

    • Building an AWS Infrastructure: Creating EC2 instances, VPC, security groups, and other essential resources using Terraform

  • Week 2 - Day 3

    Advanced Terraform with AWS

    Expand your Terraform skills on AWS by exploring advanced topics. Dive deeper into managing state, leveraging workspaces, and optimizing resource configuration. Discover advanced techniques like provisioning, lifecycle hooks, and managing secrets for secure and efficient infrastructure management.


    • Managing State: Remote state management, locking, and collaboration

    • Workspaces: Organizing infrastructure configurations for different environments (dev, staging, production)

    • Advanced Resource Configuration: Using lifecycle hooks, provisioners, and dependencies

    • Managing Secrets: Integrating with secrets management systems (AWS Secrets Manager)

  • Week 2 - Day 4

    Terraform and Google Cloud Platform (GCP)

    Explore the world of Google Cloud Platform (GCP) and learn how to provision resources using Terraform. Discover key GCP services and concepts, such as Compute Engine, Cloud Storage, and Virtual Networks. Build GCP infrastructure using Terraform and gain hands-on experience in multi-cloud deployments.


    • Introduction to Google Cloud Platform: Overview of key GCP services and concepts (Compute Engine, Cloud Storage, VPC, etc.)

    • Configuring Terraform for GCP: Authenticating with GCP, configuring GCP provider

    • Building GCP Infrastructure: Creating instances, networks, storage buckets, and other resources using Terraform

  • Week 2 - Day 5

    Terraform and Microsoft Azure

    Delve into Microsoft Azure and harness the power of Terraform for Azure resource provisioning. Understand essential Azure services like Virtual Machines, Blob Storage, and Virtual Networks. Develop the skills to create and manage Azure infrastructure using Terraform.


    • Introduction to Microsoft Azure: Overview of key Azure services and concepts (Virtual Machines, Blob Storage, Virtual Networks, etc.)

    • Configuring Terraform for Azure: Authenticating with Azure, configuring Azure provider

    • Building Azure Infrastructure: Creating VMs, storage accounts, virtual networks, and other resources using Terraform

  • Week 3 - Day 6

    Advanced Terraform Concepts

    Expand your Terraform expertise with advanced concepts. Learn dynamic resource creation using count, for_each, and dynamic blocks. Discover the power of modularization with reusable Terraform modules. Dive deeper into interpolation, output values, and leveraging Terraform Cloud for remote execution and collaboration.


    • Advanced Resource Management: Using count, for_each, and dynamic blocks for dynamic resource creation

    • Modules: Creating reusable and shareable Terraform modules

    • Interpolation and Output Values: Leveraging data from one resource in another, and extracting outputs

  • Week 3 - Day 7

    Cloud Networking

    Explore cloud networking essentials across AWS, Google Cloud, and Azure. Learn to configure subnets, routing, and security groups using Terraform. Discover load balancing and auto scaling techniques for ensuring high availability and scalability.


    • Virtual Networks: Networking concepts in AWS, Google Cloud, and Azure

    • Subnets, Routing, and Security Groups: Creating and managing networking components with Terraform

    • Load Balancing and Auto Scaling: Implementing load balancers and auto scaling groups for high availability and scalability

  • Week 3 - Day 8

    Cloud Security

    Implement security controls and best practices for a secure cloud infrastructure.


    • Managing Secrets: Integrating with secrets management systems (AWS Secrets Manager).

    • Security Best Practices: Security considerations and implementing security controls using Terraform

  • Week 3 - Day 9

    Terraform Workflows and Collaboration

    Optimize your Terraform workflows for efficient collaboration. Explore version control with Git and learn best practices for managing Terraform code. Integrate Terraform into CI/CD pipelines for automated deployments. Discover the power of collaboration and teamwork using Terraform Cloud.


    • Version Control with Terraform: Managing Terraform code with Git and best practices.

    • Continuous Integration and Deployment: Integrating Terraform with CI/CD pipelines

  • Week 3 - Day 10

    CI/CD integration and Recap

    Recap the key concepts and best practices covered throughout the training. Apply your newly acquired knowledge to hands-on projects, demonstrating your proficiency in deploying and scaling web applications in the cloud.


    • Recap of key concepts and best practices

  • Week 4 - Day 11-15

    Real-Time Project: Mastering Cloud Infrastructure with Advanced Terraform Techniques

    Take your Terraform skills to the next level with our Real-Time Project, designed to provide practical experience and boost your confidence in cloud infrastructure management. In this hands-on project, you will tackle an actual work scenario, gaining valuable insights and demonstrating your expertise to potential employers. Implement advanced tools and industry best practices learned during our Advanced Terraform Training, impressing interviewers and showcasing your ability to deploy scalable web applications across multiple cloud platforms.
