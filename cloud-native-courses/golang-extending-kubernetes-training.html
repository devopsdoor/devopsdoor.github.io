RoadMap

Your Learning Curve.

  • Week 1 - Till Day 5

    Pre-Requisites

    A practical training that covers all the basics you need. It covers

    microservices (React, JAVA and Python),
    building and running them on cloud VMs,
    containerizing and deploying them using Docker,
    Running and Orchestrating them on Kubernetes etc !

  • Week 2 - Day 1

    GET STARTED WITH GoLong - INTRODUCTION, SETUP AND FIRST PROGRAM

    Introduction to Go programming language, setting up a local dev environment, and running the first Go program.


    • Introduction to Go programming language and its features

    • Setting up a local development environment for Go programming

    • Writing, building and running a basic Go program

  • Week 2 - Day 2

    MASTER GoLong VARIABLES AND PRIMITIVES- TYPE, DECLARATION AND NAMING CONVENTIONS

    Learn about variables and their types in Go, declaration, shadowing variables, and naming conventions.


    • Understanding the various types of variables in Go programming language

    • Declaring and using variables in Go programs

    • Working with primitive data types like Boolean, Numeric, and Text types in Go

  • Week 2 - Day 3

    LEARN GoLang CONSTANTS AND ARRAYS - CREATION, BUILT-IN FUNCTIONS AND MANIPULATION

    Discover how to work with constants in Go, create arrays and slices, and learn about built-in functions.


    • Understanding constants in Go and their use cases

    • Creating and manipulating arrays in Go programming

    • Exploring built-in functions to work with arrays and other data types in Go

  • Week 2 - Day 4

    EXPLORE GoLang MAPS AND STRUCTS, AND CONTROL FLOW STATEMENTS- IF/ELSE, SWITCH, AND LOOPS

    Learn about working with maps and structs in Go, and how to use if and switch statements.


    • Understanding maps and how they can be used in Go programming

    • Creating and working with structs in Go

    • Using control flow statements like If/Else, Switch, and Loops inÂ GoÂ programming

  • Week 2 - Day 5

    DISCOVER ADVANCED GoLang CONCEPTS - DEFER, PANIC, RECOVER AND POINTERS

    Understand how to use for statements, defer, panic, and recover in Go, and how to work with pointers.


    • Understanding advanced GoLang concepts like Defer, Panic, and Recover

    • Working with pointers in Go programming

    • Exploring the nuances of working with nil and internal pointers in Go programming

  • Week 3 - Day 6

    DIVE DEEP INTO GoLang FUNCTIONS AND INTERFACES - PARAMETERS, RETURN VALUES AND BEST PRACTICES

    Gain expertise in Go functions, including basic syntax, parameters, return values, anonymous functions, and methods.


    • Understanding functions and their use cases in Go programming

    • Working with interfaces in Go programming language

    • Exploring best practices for using functions and interfaces in Go programming

  • Week 3 - Day 7

    HARNESS THE POWER OF GoLang CONCURRENCY - GOROUTINES, CHANNELS AND SELECT STATEMENTS

    Learn about Go goroutines, creating them, synchronization, parallelism, channels, and select statements.


    • Understanding the basics of GoLang concurrency and its importance

    • Creating and managing Goroutines in Go programming

    • Working with channels and select statements for advanced concurrency in Go programming

  • Week 3 - Day 8-9

    GET HANDS-ON EXPERIENCE WITH GoLang AND DATABASE SCHEMA

    Understand how to perform database schema migrations and observability use cases in Kubernetes.


    • Understanding the basics of database schema migrations and their importance

    • Building hands-on experience with GoLang and database schema migrations

    • Developing an understanding of how GoLang can be used to manage and migrate databaseÂ schemas

  • Week 3 - Day 10-15

    BUILD ADVANCED PROJECTS AND KUBERNETES USING GoLang

    Gain hands-on experience with advanced Kubernetes features by working on an extended Kubernetes project.


    • Building advanced projects using GoLang and Kubernetes

    • Exploring the ways in which GoLang can be used to extend and enhance Kubernetes

    • Developing an understanding of advanced Kubernetes features and how they can be usedÂ withÂ GoLang.
